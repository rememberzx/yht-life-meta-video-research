## 跨素材结论
1. 开头必须先解决“为什么现在看”：痛点、反常识、速度承诺三选一。
2. 中段用成分/机制做理性兜底，减少“智商税”疑虑。
3. 结尾 CTA 与风险逆转（折扣/保障）绑定，显著提高行动率。
4. 新品牌更适合 UGC 质感与高密度字幕，而非重制作品牌片。

## 首轮脚本测试建议（给 yht-life）
1. 15-25秒痛点快切版：失眠/焦虑症状 -> 核心机制一句话 -> 当晚可试 CTA。
2. 30-45秒机制说明版：反常识开头（不用XX）-> 成分映射 -> 真实使用场景。
3. 20-35秒亲和UGC版：第一人称挑战（今晚试试）-> 次日状态反馈 -> 优惠/保障。
